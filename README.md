# RAV-01: Tank Battle

Небольшой 2D-прототип танкового боя на Unity: игрок против одного ИИ-противника на одной арене.

## Требования

- Unity **6.5 (6000.5.2f1)** — версия зафиксирована в `ProjectSettings/ProjectVersion.txt`.
- Открывать через Unity Hub, стартовая сцена — `Assets/Scenes/SampleScene.unity`.

## Управление

| Клавиши | Действие |
|---|---|
| W / A / S / D | Движение танка (4 направления) |
| Space | Выстрел |
| Escape | Рестарт текущего боя |
| Стрелки / W-S (в меню) | Навигация по пунктам меню |
| Enter / Space (в меню) | Подтвердить выбор |

## Игровой цикл

1. При запуске сцены показывается главное меню (Start / Options / Exit), игра при этом на паузе.
2. **Start** снимает паузу — на арене уже ждут игрок и противник.
3. У игрока и у противника по 3 очка здоровья (см. HP-счётчик в левом верхнем углу). Попадание пули снимает 1 очко.
4. Когда чьё-то здоровье доходит до нуля — экран «Победа!»/«Поражение», игра ставится на паузу. Escape запускает бой заново.
5. **Options** пока ничего не делает — настраивать ещё нечего.

## Структура кода (`Assets/`)

- `Tank.cs` — управление игроком, стрельба, движение с ограничением по границам видимой камеры.
- `Enemy.cs` — простой ИИ противника: едет к игроку по одной оси за раз (как и игрок), стреляет с кулдауном.
- `Bullet.cs` — фабрика пуль (`Bullet.Spawn`), урон через `Health`, защита от «дружественного огня» по тегу владельца.
- `Health.cs` — переиспользуемый компонент здоровья с событием `OnDeath`.
- `ImpactEffect.cs` — эффект попадания на процедурных частицах (без внешней графики).
- `GameManager.cs` — спавн противника, HP-счётчик, победа/поражение, рестарт.
- `MainMenu.cs` — главное меню, собранное во время выполнения из спрайтов, нарезанных из `Assets/splash.png`.

## Сборка

Готовых сборок в репозитории нет — собирается через `File > Build Settings` в Unity Editor (`Windows, Mac, Linux Standalone`).
